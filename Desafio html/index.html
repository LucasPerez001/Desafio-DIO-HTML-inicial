Cliente na linguagem web

Cliente é o dispositivo que está consumindo a informação da web, que são recebidas através do server. Além disso, tanto a velocidade de conexão quanto o tamanho do arquivo afetarão diretamente a experiência do usuário.

Quando você utiliza um site, os arquivos dele ficam armazenados em seu computador para que você possa acessa-los de forma mais veloz na próxima visita. Estes arquivos são chamados de Arquivos Cache.

A tecla F5 irá apagar o Cache e requisitar novamente as informações do site, atualizando assim as informações mostradas ao usuário.

    • São sistemas, sites, que você acessa através do navegador. Esta nomenclatura não engloba os sistemas utilizados através de APPS

    Estrutura Básica do HTML

    Todo HTML é constituido por três partes ou TAGS principais:

        Tag HTML : Tudo que estiver dentro desta Tag será reconhecido pelo sistema como um comando HTML
        Tag Head : Também conhecido como Cabeçalho, engloba o conjunto de comandos e dados que o sistema carrega antes de exibir a página ao usuário.
        Tag Body : Tudo o que houver dentro desta tag aparecerá para o usuário na tela. Aqui também colocamos a Tag Title (Título) que aparecerá como nome na aba da Web.

    Atributos de TAGS

    Existem centenas de tags diferenes e que podemos utilizar para as mais diversas funções, entretanto há detelhes aos quais devemos ter cautela, quanto à sua estrutura e atributos.

    Quanto à estrutura, em sua maioria, as tags devem ter início e fim, sendo iniciadas por "< >" e finalizadas por "< />".

    Quanto aos atributos, para simplificar, daremos uma breve explicação sobre os 3 mais genéricos e 2 atributos mais específicos.

        id="" : Da indentidade a um elemento, que pode ser acessado ou chamado através de um comando.
        style : Usado para aplicar um comando css em um ou mais elementos
        class : Define uma classe de objeto.
        Input : Cria um campo texto ou numérico no HTML
        img : Coloca uma imagem no HTML

    Textos(Tipografia)

    Neste tópico discorreremos acerca de algumas tags utilizadas na construção textual do nosso arquivo Html.

    Tags

    1. H1: Nesta Tag colocaremos o título do texto
    2. H2, H3, H4, H5, H6: Nestas tags colocaremos os subtítulos do texto. Cada subitítulo está em um nivel hierárquico mais baixo que o anterior.
    3. (< p >) : Esta Tag serve para iniciar um novo parágrafo.
    4. < BLOCKQUOTE >: Tag utilizada para fazer citações no texto.
    5. < sup >: Escreve acima da linha.

    Listas Ordenadas e não Ordenadas

    Listas ordenadas (OL)

    Os itens inseridos em uma lista ordenada (ol) serão colocados na lista em uma ordem numérica crescente. Além disso, para inserirmos os itens dentro de uma lista devemos utilizar a tag chamada LI, onde podemos inserir outra tag OL e criar uma outra lista dentro da lista anterior.

    Listas Não Ordenadas (ul)

    Ao contrário de uma lista ordenada, uma lista não ordenada irá organizar os itens inseridos nela através de marcadores como ponto, vírgula, hashtag, asterísco, etc.

    Links

    Existem algumas formas diferene com as quais podemos incluir liks dentro do nosso site, neste sentido podemos tano incluir liks que levarão o usuário a paginas e sites diferentes quanto links que nos farão navegar dentro da nossa própria pagina, simplificando e tornando a navegação mais prática e agradável.

    A tag que utilizamos para isso é HREF, nela podemos incluir o endereço da pagina, arquivo ou site ao qual queremos direcionar o nosso leitor. Uma ferramenta extremamente útil para diversas ocasiões e que deixará o nosso arquivo HTML ainda mais atrativo e interaivo.
